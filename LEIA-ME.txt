TOTAL OBRAS — DEMONSTRAÇÃO FIEL À V3.3 BETA 2.2
================================================

TESTAR DIRETAMENTE NO COMPUTADOR
--------------------------------
1. Extraia todo o ZIP para uma pasta normal.
2. Abra demo-local.html no Chrome ou Edge.
3. Esse arquivo reúne aplicação, CSS e JavaScript em um único HTML.
4. Não abra o arquivo diretamente de dentro do ZIP.

PUBLICAR NO GITHUB PAGES
------------------------
1. Envie TODOS os arquivos e a pasta assets para a raiz do repositório.
2. O arquivo inicial do site é index.html.
3. O CNAME já aponta para demo.totalobras.com.br.
4. Execute:

   git add .
   git commit -m "Publica demo fiel Total Obras Beta 2.2"
   git push
