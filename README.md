# Total Obras — Demo em 5 etapas com custos adicionais

Demonstração estática do fluxo de criação de orçamento do Total Obras.

## Abrir localmente

Extraia o ZIP e abra `demo-local.html` no Chrome ou Edge.

## Publicar no GitHub Pages

Envie para a raiz do repositório:

- `index.html`
- `404.html`
- `demo-local.html`
- `CNAME`
- `.nojekyll`

## Atualização desta versão

A etapa 3 agora separa:

- honorários profissionais;
- custos adicionais;
- subtotal de honorários;
- subtotal de custos;
- subtotal da proposta;
- desconto e total final.

Os custos adicionais aceitam descrição, quantidade, unidade, valor unitário e observação. Eles aparecem na prévia, ficam salvos no orçamento e são exibidos separadamente no PDF demonstrativo.

A marca d'água da demonstração continua obrigatória no PDF.
