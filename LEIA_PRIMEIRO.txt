TOTAL OBRAS — DEMONSTRAÇÃO COMPLETA

1. Extraia todo o ZIP.
2. Abra index.html no Chrome ou Edge.
3. A tela Modelos de proposta contém os descritivos completos.
4. Em Orçamentos, clique em Visualizar para abrir o documento montado.
5. Em Editar, selecione um modelo e altere objeto, escopo, metodologia, prazo, pagamento, exclusões e observações.
6. Para publicar, substitua os arquivos do repositório da demo e faça git add ., git commit e git push.

A demonstração usa somente dados fictícios e armazenamento local do navegador.
